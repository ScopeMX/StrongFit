# StrongFit
## Instalacion

Los siguientes comandos deberian crear el proyecto y tenerlo listo para correrlo:

```sh
$ pip install -r requirements.txt
```
Es necesario tener la base de datos ya creada para poder correr el proyecto

```sh
$ CREATE DATABASE <dbname> CHARACTER SET utf8;
```

Para correr el proyecto

```sh
$ manage.py runserver
```
